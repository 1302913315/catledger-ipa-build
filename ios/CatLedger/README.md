# 猫猫记账

粉粉嫩嫩风格的 iPhone 本地离线记账 App。项目使用 SwiftUI 编写，数据保存在 App 的 Documents 目录中，不需要账号和后端服务。

实现说明：当前版本使用集中式 `LedgerStore` + 本地 JSON 快照持久化，方便在没有 Mac/Xcode 的 Windows 环境中交付完整源码。之后如果要迁移到 SwiftData，可以保留界面和模型语义，替换 `LedgerStore` 的读写实现。

## 功能

- 首页：本月收入、支出、结余、预算进度、最近账单。
- 记账：收入、支出、转账，支持分类、账户、日期、备注、标签、退款和报销标记。
- 明细：搜索、筛选、编辑、删除。
- 统计：近 7 日趋势、分类占比、消费排行榜、账户余额。
- 预算资产：总预算、分类预算、资产、负债、净资产。
- 工具：小票 OCR、CSV 导入、CSV 导出、JSON 本地备份与恢复。
- 我的：分类管理、账户管理、猫咪头像展示。

## 打开方式

1. 在 Mac 上安装 Xcode 15 或更新版本。
2. 打开 `CatLedger.xcodeproj`。
3. 在 Signing & Capabilities 中选择你的 Apple ID 或开发者团队。
4. 选择 iPhone 模拟器运行，或连接 iPhone 后选择真机运行。
5. 真机运行时，iPhone 可能需要开启“开发者模式”。

当前 Windows 环境不能生成可直接安装到 iPhone 的签名 IPA。需要 Mac + Xcode 才能编译和安装。

## CSV 导入格式

第一行可以是表头，后续每行格式如下：

```csv
日期,类型,金额,分类,账户,备注,标签
2026-06-07,支出,36.50,餐饮,微信,午饭,工作日
2026-06-07,收入,500.00,兼职,银行卡,设计稿,副业
```

说明：

- 日期建议使用 `yyyy-MM-dd`。
- 类型支持 `支出`、`收入`、`转账`。
- 分类和账户名称会匹配 App 内已有项目；没匹配到分类时会使用同类型默认分类。
- OCR 和导入都需要用户确认后才会入账。
